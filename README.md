# nbcg
Promote and discuss standards to make no build web development easier.

## Getting Involved
Join the [No Build W3C Community Group](https://www.w3.org/community/nobuild/). 
This group is open to any developer with an interest in an approach to frontend that avoids over-engineering, dependencies / NPM installs and build tooling. 
Head to the Discussions tab where we will discuss what features of HTML, CSS, and JavaScript will lead to an improved experience for those developers that value simplicity. 

## Scope of Work
The mission of this group to push for web standards that will lead to a better experience for developers that create websites with either no build tools or minimal build tools. We will assess developer needs and identify gaps, limitations, or interoperability issues in existing web platform capabilities. We will explore and help validate emerging web standards help validate emerging web platform features and communicate our needs to groups such as the WHATWG, CSSWG, W3C Working Groups, and TC39.
